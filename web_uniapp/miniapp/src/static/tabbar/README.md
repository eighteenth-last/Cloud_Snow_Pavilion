# TabBar 图标说明

请将以下8个图标文件放入此目录：

## 未选中状态（灰色）：
- home.png - 首页图标（建议尺寸：81x81px）
- menu.png - 菜单图标（建议尺寸：81x81px）
- order.png - 订单图标（建议尺寸：81x81px）
- user.png - 我的图标（建议尺寸：81x81px）

## 选中状态（绿色 #3cc51f）：
- home-active.png - 首页图标选中态
- menu-active.png - 菜单图标选中态
- order-active.png - 订单图标选中态
- user-active.png - 我的图标选中态

## 图标设计建议：
- 尺寸：81x81px（或162x162px @2x）
- 格式：PNG，支持透明背景
- 风格：简约线性图标
- 颜色：未选中 #999999，选中 #3cc51f

## 参考图标设计：
- 首页：房子/主页图标
- 菜单：菜单列表/餐饮图标
- 订单：订单列表/文档图标
- 我的：用户头像/个人中心图标

临时使用前，可以从iconfont.cn下载免费图标。
